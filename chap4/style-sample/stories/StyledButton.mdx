## StyledButton
StyledButtonはカスタムコンポーネントです。'variant'でボタンの色を切り替えることができます。

- [Primary](#primary)
- [Success](#success)
- [Transparent](#transparent)
- [Test](#natsuha)

### Primary
```tsx
<StyledButton variant="primary">Primary</StyledButton>
```

### Success

<StyledButton variant="success">Success</StyledButton>
```

### Transparent

<StyledButton variant="transparent">Transparent</StyledButton>
```

<StyledButton variant="transparent">Transparent</StyledButton>

### Test
```tsx
<StyledButton variant="natsuha">Test</StyledButton>
```
